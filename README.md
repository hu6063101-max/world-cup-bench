# ⚽ 世界杯替补席 World Cup Bench

一款纯原生 HTML / CSS / JavaScript Canvas 像素小游戏。
你是替补球员，比赛自动进行——你的任务是趁教练不注意在场边偷偷搞事，
帮"香蕉共和国"赢下世界杯，同时千万别被教练抓到。

## 运行方式

无需安装任何东西，**双击打开 `index.html` 即可游玩**（推荐 Chrome / Edge）。

也可以用本地服务器运行：

```
python -m http.server 8000
```

然后浏览器访问 `http://localhost:8000`。

## 怎么玩

比赛 90 分钟（现实约 2 分钟）自动进行，顶部实时显示比分、比赛时间和倒计时。
你坐在替补席上，用三种小动作提高己方进球概率：

| 按键 | 操作 | 效果 |
|------|------|------|
| `1` | 🍌 扔香蕉皮 | 对方球员滑倒，化解对方进攻，己方更容易拿球 |
| `2` | 🥤 递能量饮料 | 队友加速，更容易发起进攻 |
| `3` | 📢 干扰门将 | 对方门将慌乱，己方射门更容易进 |
| `空格`（按住） | 🏃 假装热身 | 教练盯人时的保命姿势 |
| `R` | 🔄 重新开始 | 比赛结束后可用 |

页面下方也有对应按钮，手机可直接点按（热身按钮按住生效）。

## 核心规则

- **教练会随机回头！** 回头前屏幕会闪烁 **⚠ 红色警告**（伴随提示音），这是你收手的信号。
- 教练盯着替补席时：
  - 还在搞事 → **当场被抓**，怀疑值大涨；
  - 呆坐不动 → 怀疑值缓慢上涨；
  - **按住空格假装热身 → 完全安全**。
- 怀疑值涨满 100% → 被赶出球场，游戏失败。
- 教练没在看你时，怀疑值会缓慢回落。
- 比赛结束后，会根据**比分、怀疑值和搞事次数**生成一份《世界杯八卦日报》搞笑新闻。

## 技术说明

- 无任何外部依赖、无图片素材：画面全部由 Canvas 代码绘制（像素风），音效由 WebAudio 实时合成。
- 文件结构：
  - `index.html` — 页面骨架、操作按钮与说明
  - `style.css` — 像素风界面样式
  - `game.js` — 全部游戏逻辑（比赛模拟、教练 AI、怀疑值、粒子/震屏特效、新闻生成）
